# MVP.gg-Esports-Website-
MVP is a game-agnostic competitive ladder. Build your team, target opponents, report scores, and watch your ELO move in real time. Project I made for fun with the hopes of competing with  something like CMG, Console Kings, or GameBattles. The target is competitive shooter players.

# Side note 
This is not going to be the most technical, but I think some may find it interesting, if not for the problem-solving. 

I'm going to cancel the site since it costs money, so here is how I did it! I used Lovable Pro for this whole process.
